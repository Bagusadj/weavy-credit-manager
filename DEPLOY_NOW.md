## Langkah 2: Push Code

```bash
cd /home/ubuntu/auto-credit-manager

# Ganti USERNAME dengan GitHub username kamu
git remote add origin https://github.com/YOUR_USERNAME/weavy-credit-manager.git
git branch -M main
git push -u origin main
```

**Masukkan GitHub password/token kalau diminta.**

## Langkah 3: Deploy di Render

1. Buka: https://render.com
2. Sign up dengan: `bagusadj22@gmail.com`
3. Klik "New +"

5. Klik "Connect GitHub"

7. Render auto-detect config
8. Klik "Create Web Service"
9. Tunggu 2-3 menit
